# TeeTechs Landing Page 🚀

A modern and responsive landing page for a fictional web development academy called **TeeTechs**. Built using HTML, CSS, and JavaScript.

## 🌐 Live Preview


---

## 📌 Features

* ✅ Responsive design (mobile & desktop)
* 🎨 Modern UI with clean layout
* 🧭 Navigation bar with smooth scrolling
* 📦 Features section with grid layout
* 💼 Projects showcase section
* 🎯 Call-to-action buttons
* ⚡ Simple and lightweight structure

---

## 🛠️ Built With

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (for interactivity)

## 🚀 Getting Started

1. Clone the repository:

   ```
   git clone https://github.com/udungeriphymbarr/teetechs-landing-page.git
   ```

2. Open the project folder:

   ```
   cd teetechs-landing-page
   ```

3. Open `index.html` in your browser.
